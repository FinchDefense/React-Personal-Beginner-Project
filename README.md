# React-Personal-Beginner-Project
📝 Task Tracker with Filter

Build a simple task management app that demonstrates:

- Component hierarchy
- State management
- Event handling
- Conditional rendering
- Lists and keys
- Props drilling
- Basic forms



1. Component Structure

Create these components:

text
App (parent)
├── Header
├── TaskForm
├── TaskFilters
└── TaskList
    └── TaskItem (multiple)


2. Features to Implement

State Management:

Tasks array (each task: { id, text, category, completed })
Filter state (e.g., "All", "Active", "Completed")
Search term state
Functionality:

Add new tasks with category (Work, Personal, Urgent)
Toggle task completion (click on task)
Delete tasks
Filter tasks by status
Search tasks by text
Show task count (e.g., "3 tasks remaining")


3. UI Requirements

Clean, simple design (CSS provided or use basic styling)
Responsive layout
Visual feedback for completed tasks (strikethrough)
Category badges with different colors
